JSP技术

JSP文件的执行流程：
  *.jsp --> *.java -->*.class

JSP文件中Java语句用<% %>格式区别，这种被称为Scriplet
 Java语句在JSP文件中共有三种形式：
    1)<% %> 主要用于局部变量、编写语句
    2)<%!% %!%> 主要用于定义全局变量、方法、类
    3)<%= %> 主要用于表达式输出
    另外,<% %>这种形式也可以用scriptlet标签<jsp:scriptlet> </jsp:script>表示
    
    
JSP注释：
  显示注释<!-- 注释  --> 
  隐示注释// 注释
        /* 注释 */
        <%-- 注释 --%>
